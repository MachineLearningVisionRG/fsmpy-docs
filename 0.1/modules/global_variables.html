Global Variables¶

__version__ = '0.1.0'[source]¶

Includes measure types variables, specifying the type of measure to use in specific measures. The types are used to specify a specific measure function proposed in a study. For example, Atanassov proposed 4 types of distances in a single study, the following constants are used to specify each distance type to calculate using the atanassov measure:

>>> DISTANCE_HAMMING
>>> DISTANCE_EUCLIDEAN
>>> DISTANCE_NORMALIZED_HAMMING
>>> DISTANCE_NORMALIZED_EUCLIDEAN

Includes the following types:

>>> DISTANCE_HAMMING = 'Hamming'
>>> DISTANCE_EUCLIDEAN = 'Euclidean'
>>> DISTANCE_NORMALIZED_HAMMING = 'normalized Hamming'
>>> DISTANCE_NORMALIZED_EUCLIDEAN = 'normalized Euclidean'
